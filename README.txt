
Це бібліотека з відкритим кодом написана на C# призначена для створення програм для обліку, 
або для інших програм яким потрібна база даних.
---------------------------------------------------------------------------------------------------------------------------
Конфігуратор
	Конфігуратор - це окрема програма яка дозволяє маніпулювати обєктами конфігурації.
	Обєкти конфігурації - це довідники, документи, перелічення, регістри і т.д

Конфігурація
	Конфігурація зберігається в ХМЛ файлі. 
	При старті платформи завантажується конфігурація.
	Модифікація конфігурації з конфігуратора. 
	Програмування з Visual Studio.

База даних
	PostgreSQL https://www.postgresql.org/
	Драйвер https://www.npgsql.org/
	Адмінка https://www.pgadmin.org/
